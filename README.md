# 🔐 NQTXT Secure Editor

A simple AES‑encrypted text editor built with Python and Tkinter.  
Supports a custom `.nqtxt` secure file format protected by a user‑defined password.  
Designed for storing private notes, confidential information, and personal data.

![image alt](https://github.com/niqonoroff/aes-secure-text-editor/blob/main/screenshot.png?raw=true)

## ✨ Features

* AES‑256 encryption for all saved files  
* Password‑protected `.nqtxt` custom format  
* Simple and clean Tkinter-based interface  
* Create, open, edit and save encrypted notes
* Built‑in calculator that opens in a separate window
* Ready-to-build standalone executable (PyInstaller)
